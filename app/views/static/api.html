GoldMaps API Documentation


Get All Nodes

https://goldmaps.herokuapp.com/api/v1/nodes

Sample Response: 

  [
    {
      "id":3,
      "room_name":"ANIME WAS A MISTAKE",
      "x":"0.0",
      "y":"2.0",
      "room_capacity":200,
      "room_number":"1.0",
      "floor":7,
      "building":"GOLD",
      "children":[]
    },

    {
      "id":2,
      "room_name":"FAHIM PLS",
      "x":"10.0",
      "y":"0.0",
      "room_capacity":4,
      "room_number":"200.0",
      "floor":2,
      "building":"EDU",
      "children":[]
    }
  ]
Returns all nodes in database with attributes in JSON format


Accepts Params:


Floor:

- Takes integer value for floor.

- For example https://goldmaps.herokuapp.com/api/v1/nodes?floor=1, which returns all nodes on first floor.


Building:

- Takes string value for building.

- For example https://goldmaps.herokuapp.com/api/v1/nodes?building=GOLD, which returns all nodes in GOLD building.


Accepts Multiple Params:

- For example https://goldmaps.herokuapp.com/api/v1/nodes?building=GOLD&floor=1, returns all nodes on first floor in building GOLD.



Get A Node

https://goldmaps.herokuapp.com/api/v1/nodes/2

Sample Response: 

    {
      "id":2,
      "room_name":"FAHIM PLS",
      "x":"10.0",
      "y":"0.0",
      "room_capacity":4,
      "room_number":"200.0",
      "floor":2,
      "building":"EDU",
      "children":[]
    }
Returns a node from the database with node attributes in JSON format


Accepts Params:


Children:

- Takes boolean value for children param, True/False.

- For example https://goldmaps.herokuapp.com/api/v1/nodes/4?children=true, which only returns all the children on parent node.
